隐藏UITableView多余的分割线

发表于 2016-04-07 | 分类于 iOS |

当tableView的style设置为plain类型时，当现实的数据不满一个屏幕的时候tableView线面有很多的分割线真是巨丑，于是就上网找解决办法，查到可以通过设置tableView的tableFooterView属性来隐藏分割线。

self.tableView.tableFooterView = [[UIView alloc]init];

设置过之后发现并没有什么卵用……难道是我人品不行吗😓。后来又在网上继续找了好长时间原来当tableView上有数据，但数据少使用上边的代码是可以实现的。但是当tableView的dataSource为空的时候，就是tableView上没有任何数据显示，这个方法是没有什么鸟用的……但是可以通过在UITableViewDataSource代理方法里边通过判断数据源，设置tableView分割显得样式就可以实现啦！！

if (!数据数组.count) { // 当个数为0的时候

    tableView.separatorStyle = UITableViewCellSeparatorStyleNone;
} else { // 当个数不为0

    tableView.separatorStyle = UITableViewCellSeparatorStyleSingleLine;
}
