UIWebView加载本地JS和CSS文件

发表于 2016-03-12 | 分类于 iOS |

项目需求

最近开发的一个项目，需要一个webView加载本地HTML文件，发现webView可以想浏览器一样识别HTML的外部CSS链接文件，虽然是很简单的上的问题，仅记录下，万一哪天老年痴呆病犯了呢……

<html> 
    <head>
    <title>My Awesome Page</title>
    <link rel="stylesheet" href="content.css" type="text/css"/>     /** 加载本地CSS文件 */
    /** href="本地文件" */
    </head>

    <body>

        ...
        <script type = "text/javascript"src = "load.js"></script>     /** 加载本地JS文件 */
    </body> 
</html>
